# Tasty-Chat-Assistant

Project: Tasty Chat Assistant

What it does: It is a chat assistant that will tell the user what a food, dish, or drink tastes like. It also reports of some allergens and will recomend foods and their tastes based on what your saved tastes are. 

Link to app: https://tasty-chat-assistant--CedricSchlecht.replit.app

Stack: (Python, Open AI API, Replit)
